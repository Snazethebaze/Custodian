# Changelog

All notable changes to Custodian are documented here. This project follows
[Semantic Versioning](https://semver.org/).

## [2.0.0] - 2026-07-18

First public release.

### Features
- **Cross-class HUD** — resource bars, buff/imbue reminders, and pre-combat checks,
  tuned per class and spec. Works across every class and power type.
- **Three tracking modes**, explained in-panel: **Live** (read moment-to-moment),
  **Pre-combat** (things hidden from addons in combat — checked before the pull),
  and **Manual** (estimated from your casts when nothing can be read).
- **Guided "Add widget" flow** — pick a resource, a maintained buff, a weapon oil,
  or search for any spell; filter the list by spec.
- **Bars & icons** you can bind to anything, drag together into groups, and move as one.
- **Colour stops** — recolour a bar as it fills and optionally ping you at a threshold.
- **Segmented display** — combo points, runes, Maelstrom stacks, and charges as boxes/pips.
- **Reminders** — show a widget only when a buff is missing, a cooldown is ready, a value
  crosses a threshold, only while in a form, or only in combat.
- **Share setups** — export any widget, folder, or class as a string and import it back.
- **Sounds & TTS**, a bundled texture/sound pack, and a movable minimap button.
